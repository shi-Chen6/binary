进制转换练习题答案

二进制转十进制

  1. 请将二进制数 10110 转化为十进制数。
    2⁴ 2³ 2² 2¹ 2⁰ (2)
    1 0 1 1 0
    = 2⁴×1 + 2³×0 + 2²×1 + 2¹×1 + 2⁰×0
    = 16 + 0 + 4 + 2 + 0
    = 22 (10)
  2. 请将二进制数 111011 转化为十进制数。
    2⁵ 2⁴ 2³ 2² 2¹ 2⁰ (2)
    1 1 1 0 1 1
    = 2⁵×1 + 2⁴×1 + 2³×1 + 2²×0 + 2¹×1 + 2⁰×1
    = 32 + 16 + 8 + 0 + 2 + 1
    = 59 (10)
  3. 请将二进制数 100001 转化为十进制数。
    2⁵ 2⁴ 2³ 2² 2¹ 2⁰ (2)
    1 0 0 0 0 1
    = 2⁵×1 + 2⁴×0 + 2³×0 + 2²×0 + 2¹×0 + 2⁰×1
    = 32 + 0 + 0 + 0 + 0 + 1
    = 33 (10)

八进制转十进制

  1. 请将八进制数 753 转化为十进制数。
    8² 8¹ 8⁰ (8)
    7 5 3
    = 8²×7 + 8¹×5 + 8⁰×3
    = 448 + 40 + 3
    = 491 (10)
  2. 请将八进制数 342 转化为十进制数。
    8² 8¹ 8⁰ (8)
    3 4 2
    = 8²×3 + 8¹×4 + 8⁰×2
    = 192 + 32 + 2
    = 226 (10)
  3. 请将八进制数 170 转化为十进制数。
    8² 8¹ 8⁰ (8)
    1 7 0
    = 8²×1 + 8¹×7 + 8⁰×0
    = 64 + 56 + 0
    = 120 (10)
  4. 请将八进制数 56 转化为十进制数。
    8¹ 8⁰ (8)
    5 6
    = 8¹×5 + 8⁰×6
    = 40 + 6
    = 46 (10)

十六进制转十进制

  1. 请将十六进制数 A3F 转化为十进制数（A代表10，F代表15）。
    16² 16¹ 16⁰ (16)
    A 3 F
    = 16²×A + 16¹×3 + 16⁰×F
    = 2560 + 48 + 15
    = 2623 (10)
  2. 请将十六进制数 4B 转化为十进制数（B代表11）。
    16¹ 16⁰ (16)
    4 B
    = 16¹×4 + 16⁰×11
    = 64 + 11
    = 75 (10)
  3. 请将十六进制数 D8 转化为十进制数（D代表13）。
    16¹ 16⁰ (16)
    D 8
    = 16¹×13 + 16⁰×8
    = 208 + 8
    = 216 (10)
  4. 请将十六进制数 2F5 转化为十进制数（F代表15）。
    16² 16¹ 16⁰ (16)
    2 F 5
    = 16²×2 + 16¹×15 + 16⁰×5
    = 512 + 240 + 5
    = 757 (10)
