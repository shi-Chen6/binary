一、二进制转十进制

1. 二进制数 10110 转十进制

24 23 22 21 20 (2)
1 0 1 1 0
= 24×1 + 23×0 + 22×1 + 21×1 + 20×0
= 16 + 0 + 4 + 2 + 0
= 22 (10)

2. 二进制数 111011 转十进制

25 24 23 22 21 20 (2)
1 1 1 0 1 1
= 25×1 + 24×1 + 23×1 + 22×0 + 21×1 + 20×1
= 32 + 16 + 8 + 0 + 2 + 1
= 59 (10)

3. 二进制数 100001 转十进制

= 32 + 0 + 0 + 0 + 0 + 1
