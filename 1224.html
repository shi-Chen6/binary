IPv4地址二进制转十进制专项练习

一、例题演示

题目：将二进制IPv4地址 11000000.10101000.00000001.00000001 转换为十进制

解题原理：IPv4由4段8位二进制数组成，每段权重从左到右为 128、64、32、16、8、4、2、1，逐段计算求和

  1. 第一段 11000000：1×128 + 1×64 + 0×32 + 0×16 + 0×8 + 0×4 + 0×2 + 0×1 = 192
  2. 第二段 10101000：1×128 + 0×64 + 1×32 + 0×16 + 1×8 + 0×4 + 0×2 + 0×1 = 168
  3. 第三段 00000001：0×128 + 0×64 + 0×32 + 0×16 + 0×8 + 0×4 + 0×2 + 1×1 = 1
  4. 第四段 00000001：计算结果同第三段 = 1

最终十进制地址：192.168.1.1

二、课堂练习题（3道）

  1. 将二进制地址 10101010.00001111.00000000.11111111 转换为十进制
  2. 将二进制地址 11001100.10011001.01010101.10101010 转换为十进制
  3. 将二进制地址 10000000.00000000.00000000.00000001 转换为十进制

三、练习题答案

  1. 10101010.00001111.00000000.11111111 → 170.15.0.255
  2. 11001100.10011001.01010101.10101010 → 204.153.85.170
  3. 10000000.00000000.00000000.00000001 → 128.0.0.1
