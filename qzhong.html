题目1：计算机发展史
世界上第一台计算机是1946年美国研制成功的，该计算机的英文缩写名为（ ）
A.MARK-II
B.EDSAC
C.ENIAC
D.EDVAC

标准答案

C

解析

ENIAC（Electronic Numerical Integrator and Computer，电子数值积分计算机）是世界上第一台通用数字电子计算机，由美国宾夕法尼亚大学在1946年研制成功。

知识拓展 - 计算机发展史

计算机四代发展：

  • 第一代（1946-1957）：电子管计算机，体积大，功耗高
  • 第二代（1958-1964）：晶体管计算机，体积小，功耗低
  • 第三代（1965-1971）：集成电路计算机，处理能力强
  • 第四代（1972-至今）：超大规模集成电路计算机

ENIAC的主要特点： 重量30吨、占地170平方米、功耗150千瓦、每秒运算5000次、需要手工编程。

题目2：CPU的组成
CPU中不包含（ ）
A.运算器
B.控制器
C.存储器
D.高速内部缓存

标准答案

C

解析

CPU主要由两部分组成：运算器和控制器。高速内部缓存（Cache）也集成在CPU内部。存储器是独立的部件，包括内存和外存，不属于CPU。

知识拓展 - 计算机硬件体系

部件 功能 说明
CPU 运算和控制 包含运算器、控制器、缓存
内存 临时存储 RAM：断电丢失；ROM：断电保留
外存 永久存储 硬盘、SSD、光盘等
题目3：CPU主频的含义
微型计算机中，关于CPU的"P4 2.4G"配置中，"2.4G"表示CPU的（ ）
A.字长
B.主频率
C.核心数
D.型号

标准答案

B

解析

"2.4G"表示主频为2.4GHz，即CPU的时钟频率为每秒24亿次。主频越高，CPU在单位时间内执行的指令越多，计算机性能相对越好。

知识拓展 - CPU性能指标

影响CPU性能的主要指标：

  • 主频（Clock Speed）：通常以GHz为单位，表示每秒钟的周期数
  • 字长（Word Length）：CPU一次能处理的数据位数（32位、64位等）
  • 核心数（Cores）：多核处理器能并行处理更多任务
  • 缓存容量：L1/L2/L3缓存影响数据访问速度
  • 工艺制程：7nm、5nm等制程越小，集成度越高
题目4：存储容量换算
1GB的准确值是（ ）
A.1024×1024 Bytes
B.1024 KB
C.1024 MB
D.1000×1000 KB

标准答案

A 或 C

解析

计算机存储单位采用二进制进制换算：

  • 1 Byte = 8 bits
  • 1 KB = 1024 Bytes
  • 1 MB = 1024 KB
  • 1 GB = 1024 MB = 1024×1024 KB = 1024×1024×1024 Bytes

所以A项1024×1024 Bytes = 1MB，应该是1024×1024×1024 Bytes；但C项1024 MB正确等于1GB。

知识拓展 - 存储容量换算

常用换算公式：

1 GB = 1024 MB 1 MB = 1024 KB 1 KB = 1024 Bytes 1 Byte = 8 bits 例：256 MB = ? GB 256 MB ÷ 1024 = 0.25 GB

注意： 不同行业的定义不同。IT行业通常用1024进制，但国际标准（IEC）中1GB=1000MB。购买硬盘时经常出现"缺容"现象，就是因为厂商用1000进制标注，而操作系统用1024进制显示。

题目5：ASCII编码
下列关于ASCII编码的叙述中，正确的是（ ）
A.一个字符的标准ASCII码占一个字节，其最高二进制位总为1
B.所有大写英文字母的ASCII码值都小于小写英文字母'a'的ASCII码值
C.所有大写英文字母的ASCII码值都大于小写英文字母'a'的ASCII码值
D.标准ASCII码表有256个不同的字符编码

标准答案

C

解析

ASCII编码是美国信息交换标准代码。标准ASCII码用7位表示，共128个字符。

  • ✗ A项：标准ASCII只用7位，最高位为0；扩展ASCII才用8位，最高位为1
  • ✗ B项：大写字母A-Z的ASCII值为65-90，小写a的ASCII值为97，所以大于a
  • ✓ C项：所有大写英文字母（65-90）的ASCII值都小于97（a的值）
  • ✗ D项：标准ASCII只有128个，扩展ASCII有256个

知识拓展 - ASCII编码表

字符类别 范围 示例
控制字符 0-31 回车('\r')=13, 换行('\n')=10
空格和符号 32-47 空格=32, '!'=33
数字 48-57 '0'=48, '9'=57
大写字母 65-90 'A'=65, 'Z'=90
小写字母 97-122 'a'=97, 'z'=122

记忆技巧： 32是空格，48是'0'，65是'A'，97是'a'。

题目7：计算机病毒的定义
计算机病毒（ ）
A.不会对计算机操作人员造成身体损害
B.会导致所有计算机操作人员感染致病
C.会导致部分计算机操作人员感染致病
D.会导致部分计算机操作人员感染病毒，但不会致病

标准答案

A

解析

这是一道考查对计算机病毒本质的理解的题目。计算机病毒是指编制者在计算机程序中插入的破坏性代码。它的定义特点：

  • ✓ 不会对操作人员造成身体损害（这是与生物病毒的区别）
  • ✗ 不是对"人"的感染，而是对"计算机"的感染
  • ✗ 影响的是计算机系统和数据，不是人的身体

知识拓展 - 计算机病毒与恶意软件

计算机病毒的特点：

  • 具有传播性：能通过网络、移动存储设备等渠道传播
  • 具有破坏性：能修改、删除、暴露数据，破坏系统
  • 具有隐蔽性：通常隐藏在其他程序中

常见类型：

  • 引导区病毒：感染硬盘引导区
  • 文件病毒：附加于可执行文件
  • 宏病毒：在Office文档中传播
  • 蠕虫病毒：能自我复制传播
  • 特洛伊木马：伪装成正常程序
题目8：CPU与内存的关系
下列关于CPU的叙述中，正确的是（ ）
A.CPU能直接读取硬盘上的数据
B.CPU能直接与内存储器交换数据
C.CPU主要组成部分是存储器和控制器
D.CPU主要用来执行算术运算

标准答案

B

解析

CPU与内存之间通过总线直接连接，能进行高速的数据交换。硬盘是外存，数据必须先加载到内存中，CPU才能读取。

知识拓展 - 计算机数据流向

硬盘(外存) → 内存(RAM) → CPU → 内存 → 硬盘 执行过程： 1. 程序从硬盘加载到内存 2. CPU从内存读取指令和数据 3. CPU执行运算 4. 结果写回内存 5. 结果保存到硬盘
题目9：I/O设备的定义

题目10：网络文明建设
在网络信息时代，青少年要学会合理利用网络信息资源，自觉抵制网络的不良诱惑，下列做法中不可取的是（ ）
A.遵守道德和法律，文明上网
B.不浏览内容不健康的网站
C.不随意答应网友的见面要求
D.为避免不良诱惑，拒绝上网

标准答案

D

解析

这是一道考查辩证思维的题目。正确的态度应该是"合理利用"而不是"拒绝"，应该学会辨别和防护，而不是因噎废食。

  • 不要在公开场合透露个人隐私信息
  • 不要随意点击陌生链接或下载未知文件
  • 定期修改密码，使用强密码
  • 安装杀毒软件并定期更新
  • 不要将重要信息保存在公共计算机上
  • 谨慎识别网络诈骗和不良内容

题目3：ASCII编码排序

答案

8 < Z < a（或 '8'、'Z'、'a'）

解析

  • '8'（数字8）= 56

学习建议：

  • 🎯 理论掌握：深入理解计算机的内部工作原理，而不仅仅记住名词
  • 📊 进制转换：这是计算机原理的基础，需要反复练习直至熟能生巧
  • 💻 实践操作：在电脑上查看硬件信息、进制转换工具等，加深印象
  • 🔍 区分概念：注意相似概念的区别，如RAM vs ROM、CPU vs 计算机等
  • ⚖️ 伦理素养：培养网络安全意识和信息素养，是信息时代必备技能

易错点提示：

  • • CPU由运算器和控制器组成，不包含存储器
  • • 主频高不一定性能就好，还要考虑架构、缓存等因素
  • • 存储容量大不等于运算速度快
  • • 标准ASCII只有128个字符（不是256个）
  • • ROM是只读存储器，ROM断电后数据不丢失
  • • 进制转换要注意16进制的字母表示（A-F）
💡 考试技巧： 遇到"一定""绝对"等绝对化词汇的判断题时，往往需要反向思考。对于单选题，先排除明显错误的选项，再在剩余选项中比较。进制转换题要养成检验答案的习惯，用转换后的结果反推验证。
